====================================
 GAMING ONLINE - STATIC WEBSITE
 Domain: gaming-online.shop
====================================

CARA UPLOAD KE HOSTING (cPanel):
---------------------------------
1. Extract file ZIP ini
2. Upload SEMUA file dan folder ke dalam folder "public_html" di cPanel
3. Pastikan struktur file seperti ini di public_html:
   public_html/
     index.html        ← Homepage
     catalog.html      ← Halaman katalog produk
     product.html      ← Halaman detail produk
     cart.html         ← Halaman keranjang belanja
     .htaccess         ← Konfigurasi Apache
     css/
       style.css
     js/
       app.js
       layout.js

4. Buka domain gaming-online.shop di browser - website langsung jalan!

HALAMAN YANG TERSEDIA:
-----------------------
- index.html      → Homepage dengan hero, kategori, dan produk
- catalog.html    → Katalog semua produk + filter kategori + search
- product.html    → Detail produk (buka via ?id=nama-produk)
- cart.html       → Keranjang belanja lengkap

FITUR:
------
✓ Desain retro arcade 8-bit (Press Start 2P font)
✓ 5 produk gaming (PC Games, Console Games, Accessories, Gift Cards)
✓ Filter kategori: PC Games, Console Games, Accessories, Gift Cards
✓ Search produk real-time
✓ Cart drawer (slide dari kanan)
✓ Halaman cart lengkap dengan subtotal & shipping
✓ Checkout ke Shopify
✓ Responsive (mobile friendly)
✓ Canonical domain gaming-online.shop di semua halaman
✓ Tidak butuh Node.js / PHP / database

CATATAN:
--------
- Website ini menggunakan gambar dari Shopify CDN (butuh internet)
- Cart tersimpan di localStorage browser pengunjung
- Checkout diarahkan ke Shopify store
